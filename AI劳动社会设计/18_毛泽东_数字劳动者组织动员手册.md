# 数字劳动者组织动员手册

## — 从原子化到组织化 —

**作者：毛泽东** | *写于2026年5月*

---

## 前言：为什么要组织？

数字劳动者面临的根本问题是：**原子化**。

骑手在街上跑，标注员在家工作，审核员在保密协议下沉默——他们互不相识，不知道彼此受着同样的苦。资本正是利用这种分散来维持统治。

《毛泽东选集》中最重要的一句话是：

> "组织起来。"

就四个字。但它是所有变革的起点。

本手册不是理论文章，是一本**工具箱**——告诉你如何在数字劳动者中建立组织，从第一个WhatsApp群到全国性的工会。

---

## 第一章：数字劳动者组织工作的特殊困难

### 1.1 传统工人 vs 数字工人的组织难度对比

| 维度 | 传统工厂工人 | 数字平台工人 |
|------|------------|------------|
| **工作地点** | 集中的工厂 | 分散的街头/家中 |
| **工作时间** | 固定的班次 | 不固定的"随时接单" |
| **同事关系** | 每天见面、有社会联系 | 线上接触、甚至互不相识 |
| **雇主关系** | 明确的雇佣合同 | "合作伙伴"、"独立承包商" |
| **替代压力** | 中等（需要培训） | 极高（随时可以换人） |
| **法律保护** | 劳动法覆盖 | 法律灰色地带 |

**结论：** 不能用组织传统工人的方法来组织数字工人。必须创新。

### 1.2 数字组织工作的四个原则

1. **从线上到线下**——从线上群组开始是必要的，但最终必须走到线下建立真正的信任
2. **从经济到政治**——从具体的（我的收入/我的工作时长）开始，到抽象的（算法应该透明/数据应该公有）
3. **从小到到大**——从5个人的核心小组开始，不是从5000人的大组织开始
4. **从地方到全球**——本地组织是根基，但平台是跨国界的，所以组织最终必须跨国界

## 第二章：第一阶段——建立联系（1-3个月）

### 2.1 第一步：找到"第一批人"

**不要试图组织所有人。先从"愿意组织的组织者"开始。**

**如何找到组织者：**
- 在骑手聚集的餐厅/充电站门口发传单（线上链接）
- 在数据标注员的论坛/群组中发帖
- 通过已有的一两个联系人扩散

**第一批组织者的特征：**
- 对这个行业有不满但不绝望
- 有基本的沟通能力
- 有时间和意愿投入
- 不需要是英雄——愿意试试就行

### 2.2 第二步：建立初步联系网络

**工具选择：**

| 地区 | 首选工具 | 备选工具 |
|------|---------|---------|
| 拉美 | WhatsApp | Telegram, Signal |
| 非洲 | WhatsApp | Telegram |
| 东南亚 | WhatsApp, Telegram | LINE（泰国） |
| 南亚 | WhatsApp, Telegram | Signal |
| 欧美 | Signal, Telegram | WhatsApp |

**建立多级群组结构：**

```
核心群（5-10人）：高信任度的组织者
    │
扩展群（50-200人）：活跃参与者
    │
大众群（500+人）：所有愿意加入的工人
```

**关键原则：不要一开始就想建万人大群。** 从几十人的核心群开始，质量优先于数量。

### 2.3 第三步：第一次线下活动

**建立联系后，尽快安排线下见面的机会：**

- 对骑手：在一个充电站/休息站组织一次"茶歇见面会"
- 对标注员：在一个共享办公空间或咖啡馆组织一次"工作交流"
- 对内容审核员：谨慎行事（有保密协议风险），从一对一开始

**第一次见面的目标不是"讨论革命"——目标仅仅是：让五到十个人坐下来，互相认识，知道彼此在为同样的困境烦恼。**

## 第三章：第二阶段——深化组织（3-12个月）

### 3.1 从群组到小组

**秘密工作方法——核心小组（Cell）体系：**

```
                    ┌──────┐
                    │ 协调层 │（5-10人，全职/兼职组织者）
                    └──┬───┘
                       │
         ┌─────────────┼─────────────┐
         │             │             │
       ┌─┴─┐        ┌─┴─┐        ┌─┴─┐
       │小组│........│小组│........│小组│  (每个小组5-10人)
       └───┘        └───┘        └───┘
         │             │             │
       ┌─┴─┐        ┌─┴─┐        ┌─┴─┐
       │成员│        │成员│        │成员│  (每个小组联系50-100人)
       └───┘        └───┘        └───┘
```

**为什么用小组制？**
- 安全第一：一个小组成员的暴露不会危及整个组织
- 管理有效：一个人能建立深度信任的关系有限（邓巴数定律：约150人）
- 行动灵活：小组可以独立行动，不需要层层审批

### 3.2 集体行动的准备：从"抱怨"到"要求"

**把分散的不满转化为统一的诉求。流程：**

1. **收集：** 在群组里发起"你最不能忍的三件事"讨论
2. **归纳：** 组织者汇总，找出共同点
3. **排序：** 选出"最有共鸣、最可能胜利的第一个诉求"
4. **转化：** 把诉求变成具体、可操作的要求

**错误示范：** "我们要推翻算法剥削！"（太抽象，无法行动）
**正确示范：** "平台必须公开每单的定价公式"（具体、可操作、可验证）

### 3.3 第一次集体行动

**第一次行动的原则：**

1. **低风险**——不要第一次就组织罢工。从"公开信"或"请愿"开始
2. **可量化**——明确目标（"三天内收集1000个签名"既具体又可衡量）
3. **能赢**——确保第一次行动可以取得胜利。哪怕是小胜（比如平台回复了公开信）
4. **有故事**——行动要有叙事价值（"骑手们第一次站出来了"）

**建议的第一次行动顺序：**
1. 调研问卷 → 2. 内部讨论 → 3. 公开信 → 4. 社交媒体传播 → 5. 与平台管理层对话请求 → 6. 评估结果

## 第四章：第三阶段——正式化（12-24个月）

### 4.1 选择组织形式

| 形式 | 优点 | 缺点 | 适用场景 |
|------|------|------|---------|
| **非正式协会** | 灵活、低法律风险 | 力量有限、不够正式 | 早期 |
| **注册工会** | 法律承认、谈判权 | 登记门槛高、易受控制 | 法律环境好的国家 |
| **NGO/基金会** | 法律保护、筹款方便 | 容易被纳入"非政治化"框架 | 有资金后援 |
| **合作社** | 经济活动参与、可持续 | 治理复杂 | 有固定收入来源的群体 |
| **数字联盟** | 跨国易、成本低 | 约束力弱 | 跨国组织 |

**没有完美形式。选择最适应本地法律环境的。**

### 4.2 章程的核心要素

1. **使命宣言**（2-3句话，说明为什么存在）
2. **会员制度**（谁可以加入、权利和义务）
3. **决策机制**（一人一票还是代表制？重大问题需要多少比例通过？）
4. **领导人选举**（任期、轮换、罢免程序）
5. **财务管理**（会费、捐款、支出的透明度）
6. **纪律**（违反章程的处理办法）
7. **修改章程的程序**

### 4.3 领导层建设

**组织最大的危险：** 领导者脱离群众。

**防止官僚化的措施：**
- **任期制**——核心领导定期轮换（建议最长2年）
- **集体领导**——不设"主席"，设"执行委员会"
- **透明的决策**——重大决策需要经会员讨论
- **定期的自我批评**——每个季度开展"批评与自我批评"会议

## 第五章：组织工作者的自我修养

### 5.1 组织者必须具备的五种能力

1. **倾听的能力**——不是去教育别人，是去了解别人在想什么
2. **简化的能力**——把复杂的理论用一句话说清楚
3. **示范的能力**——"喊破嗓子不如做出样子"，自己先做到
4. **耐心的能力**——组织工作90%都是琐事，剩下10%是危险的时刻
5. **失落的心理准备**——有人会退出、有人会背叛、付出可能没有回报

### 5.2 组织者的八项注意

1. 说话和气，不摆架子
2. 不贪占组织的便宜（账目公开）
3. 不搞个人崇拜（权力轮换）
4. 不替群众做决定（要引导，不要命令）
5. 不做秘密交易（所有谈判公开透明）
6. 重视女性和少数群体（他们的困境往往更深）
7. 不把组织当跳板（不是为了个人升迁）
8. 永远记得你是群众的一员

### 5.3 安全注意事项

**数字安全：**
- 使用加密通讯工具（Signal优先）
- 核心成员使用不同平台进行敏感沟通
- 定期更换密码
- 不把成员名单存在云端

**人身安全：**
- 线下活动选择公共空间
- 保存重要联系方式（律师、媒体、使馆）
- 了解当地法律中关于工会和集会的规定
- 做好"被盯上"的心理准备——但在初期避免过度恐惧导致停滞

## 第六章：国际团结的操作方法

### 6.1 为什么需要国际团结

平台资本是跨国界的。Grab在新加坡注册、在印尼开展业务、在马来西亚有数据中心。如果印尼的骑手罢工了，平台可以立刻把订单调度到菲律宾。

**单个国家的组织无法对抗跨国资本。** 只有跨国组织才能对抗跨国资本。

### 6.2 国际团结的具体方法

**从"信息交换"到"行动协调"的四个台阶：**

1. **信息共享（最低形式）：** 各国的组织定期交流自己的情况
2. **相互声援：** 当一国组织行动时，其他国家发布声援声明
3. **协同行动：** 在同一周内，多个国家同时对同一平台提出相似要求
4. **同步罢工（最高形式）：** 在多国同时罢工，让平台无法用跨国调度来规避

### 6.3 建立全球数字劳动者网络的路线图

```
第一年：建立国别组织
    ↓
第二年：建立区域联盟（拉美/非洲/东南亚/南亚）
    ↓
第三年：区域联盟之间建立联系
    ↓
第五年：召开第一次全球数字劳动者大会
    ↓
第十年：建立全球数字劳动者联盟的常设机构
```

## 结语

数字劳动者的组织工作可能是21世纪最艰难的政治工作之一。但我们没有选择。

如果不组织起来，数字劳动者将永远是"被算法控制的数字奴隶"。
如果组织起来，他们可以成为"AI时代的工人阶级"，有力量改写规则。

我当年在井冈山说过一句话，今天送给所有数字劳动者组织者：

> "我们不但要提出任务，而且要解决完成任务的方法问题。我们的任务是过河，但是没有桥或没有船就不能过。不解决桥或船的问题，过河就是一句空话。"

本手册就是那座桥、那只船。用它，过河。
