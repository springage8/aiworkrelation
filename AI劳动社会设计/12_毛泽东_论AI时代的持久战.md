# 论AI时代的持久战

> 知识蒸馏自《论持久战》（1938年）
> 创作日期：2026年5月2日
> 理论来源：毛泽东《论持久战》《中国革命战争的战略问题》，马克思《资本论》

---

## 一、核心论题

> **"战争是政治的继续。"** ——克劳塞维茨

AI时代的变革也是如此——它不是一场技术升级，而是围绕数据生产资料所有权展开的长期政治斗争。

本文件的批驳对象：**两种错误论调**

| 论调 | 主张 | 错在哪里 |
|------|------|---------|
| **速胜论** | "AI发展这么快，资本很快就会自我崩溃——我们设计好制度等着就行" | 低估了资本的适应性和韧性。资本可以通过吸收批判、让步改善、转移矛盾等方式延长寿命。 |
| **亡国论** | "AI垄断资本已经控制了算力、数据、模型、媒体、政府——我们改变不了什么" | 夸大了敌人的力量，忽视了矛盾的发展变化。历史上的每一次重大变革在开始时看起来都是"不可能"的。 |

**本文件的核心判断：** AI时代的变革，是一场**持久战**。它不会速胜，也不会失败——只要我们采取正确的战略。

## 二、敌我力量对比的全面分析

### 2.1 敌人的优势

AI垄断资本目前掌握着压倒性的力量优势：

```
算力优势：
  全球90%以上的AI训练算力集中在不到10家公司
  训练一个大模型（如GPT-5）需耗资数十亿美元
  门槛越来越高——数据公有制的服务器根本跑不动

数据优势：
  用户行为数据被平台独占——形成"数据护城河"
  训练数据越用越多→模型越好→用户越多→数据越多——正反馈闭环

人才优势：
  全球顶尖AI人才被高薪锁定
  2025年，Google、Meta、OpenAI、Microsoft雇佣了全球70%以上的AI博士毕业生

政策优势：
  在主要国家（美国、中国），AI政策被大型科技公司的游说深度影响
  EU的AI Act是某种例外，但执行力和监管力度仍有疑问

意识形态优势：
  "技术中立论"——让AI看起来像是"客观工具"而非权力结构
  "创新优先论"——任何监管都被描绘为阻碍进步
  "市场选择论"——消费者"自由选择"使用平台，实则无替代选项
```

### 2.2 我们的优势

变革力量的优势不在于"现在"，而在于**矛盾的发展趋势**：

```
矛盾一：利润最大化 vs 消费能力
  AI替代劳动者 → 失业增加 → 消费能力下降 → AI产品的市场萎缩
  资本不可能在消灭消费者的同时维持利润
  → 这个矛盾是结构性的、不可调和的

矛盾二：数据私有 vs 数据的社会性
  数据只有在全社会范围内流动才能最大化价值
  但垄断的封闭策略使数据被"囚禁"在平台上
  → 私有制越来越成为生产力发展的桎梏
  → 数据公有制效率更高——这不是道德判断，是经济规律

矛盾三：AI资本的全球性 vs 国家主权
  AI跨越国界，但国家仍然是政治权力的基本单位
  → 越来越多的国家（EU、中国、拉美、印度）会采取措施限制AI垄断
  → 这不是因为"他们支持公有制"，而是"他们不想被外国资本统治"
  → 但这种力量可以被我们利用

矛盾四：不断扩大的AI替代
  被AI替代的人越多 → 不满的人越多 → 潜在的变革力量越大
  当替代率从"10%失业"接近"30%失业"——系统的合法性危机就来了
```

### 2.3 综合判断

```
当前阶段：敌人战略优势期
  → 敌人掌握主动，我们在防御
  → 主要任务不是进攻，而是积累力量

中期阶段：相持阶段
  → 数据公有制的实验证明了可行性
  → AI替代失业达到临界点（估计：15-25%失业率）
  → 社会矛盾激化，中间派开始转向
  → 我们和敌人形成力量的拉锯

远期阶段：战略反攻
  → AI垄断资本的生产关系已经无法容纳生产力发展
  → 数据公有制在经济效率上被证明优于私有制
  → 政治条件成熟，制度转型开始
```

## 三、三个阶段的具体分析

### 第一阶段：战略防御（当前 — 2030年左右）

**形势判断：** 敌强我弱，力量对比悬殊。

**核心任务：** 不是"改变制度"——那是下一阶段的任务。当前的任务是：
1. **理论准备**——完成对AI时代社会矛盾的系统分析（这个仓库正在做的工作）
2. **组织准备**——从原子化的数字劳动者中孕育出第一批有组织的细胞
3. **实验准备**——在局部地区/行业建立数据公有制的试点
4. **舆论准备**——让"数据公有制"从"激进主张"变成"可讨论的议题"

**战略方针：**
```
"敌进我退"——不要正面硬碰硬
  不要试图和OpenAI比技术，和Google比资源——打不赢
  要在敌人力量薄弱的领域寻找突破口

"敌驻我扰"——在敌人放松警惕时持续施加压力
  揭露AI垄断资本的社会成本——每次裁员、每次涨价、每次丑闻都是机会
  保持持续的理论批判——不让"AI造福人类"的叙事独占话语权

"敌疲我打"——在敌人内部矛盾激化时局部反攻
  当一个AI公司出丑闻（如数据泄露、算法歧视），就是局部反攻的时机
  不是"把一个公司搞垮"，而是"把一个议题推到公共讨论的聚光灯下"

"敌退我追"——在敌人被迫让步时巩固成果
  EU的AI Act虽然不彻底，但它是进步——要推动类似的立法
  每个争取到的"算法透明度"条款，都是未来制度变革的垫脚石
```

**关键战场：**

| 战场 | 目标 | 可操作的策略 |
|------|------|------------|
| **立法** | 在各国推动算法透明和数据权利立法 | 联合学术团体、NGO、工会，在法律框架内 |
| **司法** | 推动针对AI垄断的集体诉讼 | 支持数据标注员、零工劳动者的集体诉讼 |
| **舆论** | 改变公众对"数据私有"的默认接受 | 制造标志性事件（如ChatGPT训练数据的剥削分析） |
| **组织** | 建立第一批数字工会和数据合作社 | 从最容易组织的群体开始（外卖骑手、网约车司机） |
| **技术** | 维护和扩展开源AI生态 | 支持Llama、Bloom、HuggingFace的开源社区 |

**第一阶段预计成果：**
- 数据公有制在全球范围内成为公共辩论的议题
- 至少3-5个数字工会建立并运转
- 至少1个国家通过"算法劳动者权利法案"
- 至少1个城市启动"数据公共信托"试点
- 全球AI劳工联盟雏形形成

### 第二阶段：战略相持（2030 — 2040年左右）

**形势判断：** 敌我力量对比从"悬殊"转向"接近均衡"。

**触发条件：**
- AI替代失业率达到15-25%临界点——社会不满积累到系统层面
- 数据公有制的试点证明了经济可行性——不是一个"乌托邦"，而是一个"可行的替代方案"
- AI资本内部出现分裂——部分资本意识到"数据私有制无法持续"
- 全球南方形成联盟——要求"数字主权"不再是口号而是行动

**核心任务：**
1. **从局部实验到制度推广**——数据公共信托从试点扩大到行业/城市/国家层级
2. **从防御到局部进攻**——在对我们有利的议题上（如数据隐私）推动立法
3. **从分散到联合**——各地的数字工会、数据合作社形成跨国网络
4. **从批判到建设**——不再只是"反对AI垄断"，而是"建设AI公有"

**战略方针：**
```
"大规模建立根据地"——从实验点到网络化
  数据公共信托从1个城市扩展到10个城市
  数字工会从1个行业扩展到多个行业
  开源AI生态从"爱好者社区"发展到"可替代闭源方案"的水平

"分化瓦解敌人"——利用AI资本内部矛盾
  大平台 vs 小企业——小企业不愿意被大平台的数据垄断压制
  模型公司 vs 算力供应商——英伟达的高定价让所有AI公司不满
  民用AI vs 军工AI——民用AI企业不愿被军事化带偏方向
  资本之间的裂痕，就是我们的切入点

"统一战线扩大"——把中间派拉过来
  当技术中产阶级开始被AI替代，他们的态度会从"漠不关心"转向"必须改变"
  要在他们最动摇的时候——刚被裁员时——就伸出援手
  给他们一个"不是抗争就是沉没"的选择

"局部反攻的实验"——在条件成熟的地方推进制度变革
  如果一个国家的数据公有制立法条件成熟（如EU在2030年代），全力推动
  在这一个国家的成功，会成为全球的示范效应
```

**第二阶段的风险：**

| 风险 | 描述 | 应对 |
|------|------|------|
| **资本的反扑** | 资本会动用一切力量（政治、经济、舆论）打击数据公有制试点 | 做好法律准备；保持国际团结；防止孤立被各个击破 |
| **改良主义的陷阱** | 资本可能用"假公有制"（名义公有、实控资本）来吸收改革压力 | 坚持核心原则：数据控制权必须从资本转移到劳动者和社区手中 |
| **内部分裂** | 随着运动发展，内部会出现路线分歧（激进派vs温和派） | 民主集中制——允许辩论，但一旦决定就必须执行 |
| **疲惫和倦怠** | 长期斗争消耗人的意志力 | 不断创造小的胜利——每个成功的试点、每个通过的立法，都是"加油"的机会 |

**第二阶段预计成果：**
- 数据公有制在至少5个主要国家成为主流政治议题
- 全球AI劳工联盟覆盖1000万+劳动者
- 至少10个数据公共信托在运行
- 开源AI能力接近闭源水平
- 一个或多个国家通过了全面的"算法民主化"法案

### 第三阶段：战略反攻（2040 — 2050年）

**形势判断：** 我们的力量接近或超过敌人。制度转型的政治、经济、社会条件均成熟。

**触发条件：**
- AI替代失业率达到30%+——旧制度的合法性危机
- 数据公有制在经济效率上被证明全面优于私有制
- 全球合围——大多数国家加入了数据公有制框架
- AI垄断资本内部崩溃——核心人才流失、用户大规模迁移到公有平台

**核心任务：**
1. **制度转型**——将数据公有制从"实验"升级为"标准制度"
2. **全球治理**——07号方案中的"全球算法公约"从设计变成现实
3. **生产关系变革**——从"雇佣劳动+UBI"逐步过渡到"自由劳动+按需分配"

**战略方针：**
```
"全面反攻"——不再是防御性的、实验性的，而是进攻性的、制度性的
  推动全球算法公约的签署
  推动数据公有制写入国际法
  推动AI垄断资本的拆分（类似19世纪的反垄断运动）

"建立新秩序"——从"革命的力量"变成"治理的力量"
  我们的任务不再只是"反对旧制度"
  而是"建设新制度"——这需要和革命时期完全不同的技能
  从批判者转向建设者——需要懂治理、懂管理、懂妥协

"巩固胜利"——防止复辟
  旧资本不会因为一次失败就永久退出
  必须有制度性的保障——防止"数据公有制"在形式上确立、在实质上被掏空
```

**第三阶段的风险：**

最大的风险不是"打不赢"，而是"打赢了之后怎么办"。历史上太多革命在胜利后因为内部斗争而变质。

具体的防范措施——这不是本文件的任务，但必须在第三阶段开始前就准备好。

## 四、战术层面的具体问题

### 4.1 根据地：在哪里建立第一个数据公有制试点？

参见12号文件《数据公有制的井冈山道路》的详细分析。

### 4.2 组织：如何组织分散的、原子化的数字劳动者？

```
原则：

1. 从经济共同体到政治共同体
   → 先解决"多赚钱"的问题（联合叫价、集体议价）
   → 再解决"有权力"的问题（算法规则的共同制定权）
   → 经济基础决定上层建筑——不解决生计问题的组织是空中楼阁

2. 从职业团结到阶级团结
   → 先在同一职业内部组织（外卖骑手联盟）
   → 再跨职业联合（骑手+数据标注员+网约车司机）
   → 最后形成"数字无产阶级"的阶级意识

3. 从线上到线下
   → 线上组织（微信群、Telegram群）是起点
   → 但不能止于线上——见面开会、上街抗议、集体行动
   → 线上的团结是"弱连接"，线下的团结才是"强连接"

4. 从小胜利到大胜利
   → 不要一开始就提出不可能完成的目标（"推翻AI资本主义"）
   → 从"争取每一单多五毛钱"开始——赢了就是信心
   → 小胜利积累到一定程度，自然走向大目标
```

### 4.3 宣传：如何打破"技术中立论"的意识形态？

```
三大核心叙事：

叙事一："谁控制AI，AI就为谁服务"
  不是技术本身的问题，是"谁控制技术"的问题
  关键问题：AI的收益归谁？AI的决策权归谁？AI的目标谁定？

叙事二："数据是你创造的，不是平台施舍的"
  打破"用户是消费者"的幻觉——用户是生产者
  没有你的数据，就没有AI——但AI的收益没给你

叙事三："AI替代不可怕，可怕的是替代你的不是你"
  替代本身不是问题（人类历史就是不断被工具替代的历史）
  问题是：AI替代你之后，收益到哪里去了？
  如果收益归全社会——替代意味着解放
  如果收益归私人——替代意味着剥夺

    替代 + 收益私有 = 灾难
    替代 + 收益公有 = 解放
```

### 4.4 联盟：统一战线如何构建？

```
核心层：数字无产阶级
  ——零工劳动者、数据标注员、内容审核员
  ——变革的基本力量，必须优先服务他们的利益

紧密层：技术中产阶级下层
  ——初级开发者、数据分析师、IT支持
  ——可以被争取，但不能让他们主导方向

合作层：传统资本阶层（有限合作）
  ——在"反数据垄断"议题上合作
  ——但要保持距离——他们的利益和我们在根本上是冲突的

外围层：主权国家政府（特别是全球南方）
  ——他们需要数字主权，我们需要政治支持
  ——各取所需的联盟，不是基于共同理想的联盟

利用层：AI资本内部矛盾
  ——小平台vs大平台、模型公司vs算力商
  ——敌人的敌人不是朋友，但可以是战术上的利用对象
```

## 五、持久战的哲学基础

> **"武器是战争的重要因素，但不是决定的因素，决定的因素是人不是物。"**
> ——毛泽东《论持久战》

这个判断在AI时代同样适用——甚至更加适用。

AI垄断资本的优势在"物"——算力、数据、模型。我们的优势在"人"——劳动者的数量、被压迫者的愤怒、历史发展的规律。

**历史不会自动走向进步。** 生产力不会自动决定生产关系——是人、是组织起来的人、是在正确理论指导下行动的人，把可能性变成现实性。

> **"战争的伟力之最深厚的根源，存在于民众之中。"**

AI时代的变革同样如此——**变革的伟力之最深厚的根源，存在于被AI替代和影响的数十亿人之中。**
