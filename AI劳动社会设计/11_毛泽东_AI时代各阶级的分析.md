# AI时代各阶级的分析

> 知识蒸馏自《中国社会各阶级的分析》（1925年）
> 创作日期：2026年5月2日
> 理论来源：毛泽东《中国社会各阶级的分析》《实践论》《矛盾论》，马克思《资本论》

---

## 一、核心论题

> **"谁是我们的敌人？谁是我们的朋友？这个问题是革命的首要问题。"**
> ——毛泽东《中国社会各阶级的分析》（1925年）

AI时代的变革力量已经形成，但缺乏一个清晰的**阶级分析框架**。不解决这个问题，一切制度设计都是无的放矢。

现有07号和09号方案设计了一整套制度，但从未回答：**谁推动这个变革？谁反对这个变革？谁是可以争取的中间力量？**

本文件是对这一缺口的直接补充。

## 三、敌人：变革的阻力

### 第一类：AI垄断资本

**包括：** 大型AI平台的所有者、股东、核心高管层（OpenAI的Sam Altman、Google的Pichai、Meta的Zuckerberg、字节跳动的张一鸣等及其背后的风险资本）

**经济地位：** 控制AI时代最核心的三种生产资料——数据（用户行为数据池）、算力（GPU集群和数据中心）、算法（基础模型和推荐系统）

**全球年收入规模（前五大AI平台）：** 超过1.5万亿美元，相当于一个中等发达国家的GDP。

**阶级特征：**
- 垄断程度比工业时代的托拉斯更高——网络效应+数据飞轮+生态锁定形成"赢家通吃"
- 掌握分配权——决定谁被看到（推荐算法）、谁被信任（信用评分）、谁被雇佣（AI面试）
- 控制基础设施——从芯片（英伟达）到云端（AWS/Azure/GCP）到模型（GPT/Claude/Gemini）

**政治态度：** 积极反对数据公有制、算法透明化和AI再分配。利用游说、资助学术、媒体控制塑造"AI风险需要企业自我监管"的舆论。

**策略：**
- 他们是主要矛盾方面，但当前力量对比悬殊
- 直接对抗会失败——需要利用他们内部的矛盾（市场竞争、技术路线分歧）
- 列宁说过"一个帝国主义内部也有矛盾"——英伟达和其客户（OpenAI、Google）之间有利害冲突

**典型代表人物画像：**
```
Sam Altman（OpenAI CEO）：
  公开说"AI会消灭很多工作" → 但反对UBI的资金来源是AI税
  → 主张"AI利益需要广泛分配" → 但公司估值3000亿美元不分给数据提供者
  → 这是"慈善的面孔，资本的本质"

  **评价：** 不是个人品德问题，是结构性位置决定了他必须维护股东利益
```

### 第二类：军工AI复合体

**包括：** 军用AI承包商（Palantir、Anduril、雷神、洛克希德·马丁），军事AI研发机构，依赖战争预算的科研体系

**经济地位：** 全球军事AI市场预计2030年达3000亿美元。这些企业的利润率（30-50%）远超民用AI企业。

**阶级特征：**
- 商业模式依赖冲突和战争——和平不符合其利益
- 与政府深度绑定——"旋转门"机制（前国防部长→AI企业CEO→回政府制定AI政策）
- 技术伦理约束最小——军事AI没有"用户同意"的问题

**政治态度：** 坚决反对AI武器化限制、反对算法透明度要求、反对全球AI治理公约

**策略：**
- 从道义上彻底孤立（但不能期望他们转变立场）
- 揭露他们操纵政策和舆论的手段
- 关键一战：能不能阻止**致命自主武器系统（LAWS）**的全面部署

**数量估计：** 全球约500-800家核心企业，从业人员约300万（含供应链）。

## 四、中间派：可争取的力量

### 第三类：技术中产阶级 ——*变革的关键变量*

这是最复杂、也最重要的群体。需要细分：

#### 3.1 上层技术精英（算法工程师、AI研究员、产品经理）

**经济地位：**
- 年收入：硅谷30-80万美元；中国50-200万人民币；其他地区10-30万美元
- 拥有资本意义上的人力资本——市场稀缺性赋予他们议价能力
- 但**不拥有生产资料**——代码属于公司，模型属于公司，数据属于公司

**政治态度：** 摇摆的中间派
- 算法工程师崇拜"技术中立"，对政治不感兴趣
- 但在"为谁工作"的问题上出现分化——越来越多AI研究员拒绝去军工企业（Google的AI研究员因Project Maven辞职，OpenAI的安全团队集体出走）
- 如果被大规模裁员（AI替代AI程序员），态度会剧烈左转

**策略：**
- 教育他们"技术不是中立的"——福柯的知识-权力分析
- 激发他们的职业道德——"你写下的每一行代码都有社会后果"
- 利用他们和资本之间的矛盾——当资本决定用AI替代他们自己时，他们就觉醒了

**典型矛盾：**
```
2024-2026年的趋势：
  OpenAI用GPT-4替代了人类内容审核员（肯尼亚工人被解雇）
  Google用AI替代广告销售团队（3万人失业）
  Meta用AI写代码 → 程序员开始被自己的产品替代
  越来越多的AI公司用AI标注AI的训练数据 → 人类数据标注员被挤出

趋势：技术中产阶级在"替代链条"上的位置不断前移
  → 昨天替代的是工厂工人
  → 今天替代的是数据标注员和客服
  → 明天替代的可能就是程序员自己

**这恰恰是他们转向进步阵营的物质基础**
```

#### 3.2 中层技术管理者（技术总监、产品VP、CTO）

**经济地位：** 接近资产阶级——有股票期权、管理层权力、决策参与权

**政治态度：** 保守，倾向于维护现有体制

**策略：** 不是主要争取对象。不要在他们身上浪费过多资源。

#### 3.3 外围技术从业者（IT支持、数据分析师、初级开发者）

**经济地位：**
- 年收入：发达地区5-15万美元；全球南方2-5万美元
- 工作稳定性的AI风险极高——大量重复性编码、数据处理工作将被替代
- 缺乏技术精英那样的不可替代性

**政治态度：** 潜在的激进力量——他们是技术中产阶级中最可能被AI替代的群体，也是最可能接受变革主张的群体

**策略：** 重点发展对象。他们是技术中产阶级和数字无产阶级之间的连接桥梁。

### 第四类：传统资本阶层

**包括：** 非AI行业的传统资本家——制造业、零售业、房地产、传统金融

**经济地位：** 正在被AI资本侵蚀和挤压。零售被亚马逊摧毁，金融被算法交易冲击，制造业被AI自动化替代工人

**政治态度：** 复杂且矛盾
- 他们和AI垄断资本之间存在**竞争关系**——这是可以利用的矛盾
- 但他们和劳动者之间仍然是**剥削关系**——不能信任他们
- 在特定议题上可能是"暂时的同路人"——比如支持数据隐私保护（因为他们的数据也被AI平台采集）

**策略：**
- 在不损害劳动者根本利益的前提下，可以利用他们和AI资本的矛盾
- 但不能把他们当作可靠的同盟军——一旦威胁到他们自身的剥削利益，他们会立即背叛
- 具体策略：在"反数据垄断"议题上建立有限的战术联盟

**毛泽东原典对照：**

> "那动摇不定的中产阶级，其右翼可能是我们的敌人，其左翼可能是我们的朋友——但我们要时常提防他们，不要让他们扰乱了我们的阵线。"

——这与AI时代的技术中产阶级完全吻合。

## 五、朋友：变革的基本力量

### 第五类：数字无产阶级 ——*变革的主力军*

#### 5.1 零工劳动者（骑手、网约车司机、外卖员）

**全球规模：** 约5-8亿人（含兼职），增长最快的劳动群体

**经济地位：**
- 无固定雇佣关系——法律上是"独立承包人"，实际上被算法完全控制
- 无劳动保障——没有医保、养老金、带薪假
- 收入水平：低于最低工资的案例普遍存在（扣除平台抽成后）
- 无议价能力——评分低一分，派单减一半

**阶级特征：**
- 最直接、最显著地被算法剥削——每一次派单、每一条路线、每一分钟的等待时间都被算法优化到极致
- 原子化——骑手之间没有联系，互相竞争，平台用算法制造"赛车效应"
- 但开始组织起来——中国的外卖骑手罢工、美国Uber司机工会、英国Deliveroo集体诉讼

**政治态度：** 对现状极度不满，但需要一个政治方向

**策略：**
- 这是最革命的力量——他们是最被剥削的群体，没有什么可失去的
- 帮助他们组织起来——突破平台的原子化策略
- 帮助他们从"利益抗争"（提高单价）走向"制度抗争"（算法公平、数据权利）
- 典型切入点：**"算法罢工"**——集体下线、集体给差评、集体刷假数据

#### 5.2 数字体力劳动者（数据标注员、内容审核员、AI训练师）

**全球规模：** 约5000万-1亿人

**经济地位：**
- 全球分工的最低端——标注员在肯尼亚时薪2美元，在菲律宾1.5美元
- 心理创伤严重——内容审核员持续看暴力、虐童、恐怖主义内容，创伤后应激障碍率极高
- 完全隐形——用户不知道"训练ChatGPT的不是算法，是肯尼亚的廉价劳动力"

**阶级特征：**
- 最隐蔽的被剥削群体——他们的劳动是"看不见的"（伊莲·埃尔斯沃斯说的"隐形劳动"）
- 没有品牌认知——没有人知道他们为哪个模型工作了
- 可替代性最高——AI正在替代人类标注员（用AI标注AI），这对他们来说是双重压迫

**政治态度：** 最绝望的群体——目前没有集体行动的组织条件

**策略：**
- 长期目标：消除"数字血汗工厂"
- 中期目标：透明化——要求AI公司公开标注员的劳动条件和报酬
- 短期目标：他们和AI垄断资本的矛盾是不可调和的——问题是他们还没有发声的渠道

### 第六类：边缘化与被淘汰者

#### 6.1 AI替代失业者

**包括：** 被AI替代的传统行业工人——翻译、客服、初级会计、初级律师、插画师、翻译、录入员……

**规模和趋势：**
- 2023-2030年预计全球被AI直接替代的岗位：3-5亿个（高盛、麦肯锡等多家机构预测）
- 且替代速度远超历次工业革命——不是"一代人的时间"，而是"十年以内"

**经济地位：** 既失去了旧的生产资料（传统职业技能），又没有获得新的生产资料（AI工具和算力）

**政治态度：** 愤怒但方向不明——容易被民粹主义和排外情绪吸引

**策略：**
- 这是最关键、也最危险的群体
- 如果不把他们的愤怒引导到正确的方向（反对AI垄断资本），就会被引导到错误的方向（反移民、反全球化、种族主义）
- 正确方向：**"不是AI抢了你的工作，是资本利用AI抢了你的工作"**

**历史参照：**
```
工业革命时期：
  英国卢德分子砸机器 → 机器确实毁了他们的生计
  但没有组织的破坏机器 → 只带来镇压，不带来改变
  最终组织起来的工会 → 争取了八小时工作制和劳动保障

AI时代：
  不能"砸AI"（技术本身不是敌人）
  要"砸开AI的所有权"（谁控制AI才是核心问题）

  卢德分子的教训：
    他们砸的不是机器，而是资本主义制度的"物质表现"
    但他们没有区分"机器"和"机器的资本主义应用"
  → 我们要帮第六类群体区分"AI技术"和"AI的资本主义应用"
```

#### 6.2 数字难民

**包括：** 缺乏数字接入能力的人群——全球仍有26亿人没有互联网（ITU数据），老年人、低收入地区人口

**策略：** 不能把他们当作"被动的受害者"——他们不是变革的障碍，而是变革的受益者。真正包容的AI社会必须从他们的需求出发，而不是"先服务好数字用户，再考虑他们"。

## 七、阶级分析的战略应用

### 7.1 谁是主要敌人？

**当前阶段的主要敌人 = AI垄断资本**

不是"所有资本家"，也不是"AI技术本身"。明确这一点至关重要——如果把目标设定得过大（"消灭资本主义"），那在当前力量对比下完全没有可行性。如果把目标设定得过小（"让AI更公平"），那就会被资本吸收为改良主义。

正确的中期目标：**数据公有制**——这是资本主义能接受的极限（类似公共图书馆、公共教育），也是通往更深入变革的起点。

### 7.2 谁是主要同盟军？

**数字无产阶级（零工劳动者+数字体力劳动者）是基本力量。**
**技术中产阶级的下层是可以争取的主要同盟军。**
**AI替代失业者是必须争取的关键群众。**

### 7.3 谁是可以利用的矛盾对象？

**AI垄断资本 vs 传统资本**——在"数据垄断"议题上有共同利益
**AI垄断资本内部**——大平台vs小企业、模型公司vs算力供应商（英伟达）
**AI垄断资本 vs 国家主权**——尤其是在拉美、欧洲，国家有对抗美国AI霸权的动力

### 7.4 当前的行动方针

根据以上分析，当前阶段的行动方针应当是：

```
战略方向：数据公有化（不是消灭资本，而是划定资本的边界）
主要矛盾：AI垄断资本 vs 全社会（包括传统资本、国家主权）
主要力量：数字无产阶级 + 技术中产阶级下层
统一战线：传统资本中的反垄断力量 + 主权国家政府 + 全球南方
斗争形式：立法斗争（算法透明法）+ 经济斗争（数据红利）+ 组织斗争（数字工会）
```

## 九、总结：以阶级分析的眼光看AI治理

没有阶级分析的AI治理方案，就像没有地图的航行。

07号和09号方案设计了漂亮的制度蓝图——但**制度不会自己出现**。制度是阶级力量对比的产物。数据公有制不会因为"它在逻辑上正确"就自动实现——它需要有一个阶级（数字无产阶级）来推动它。

**本文件的核心结论：**

> AI时代的变革，不只是一个"技术治理"问题，也不是一个"伦理设计"问题——它首先是一个**政治问题**。而政治问题的核心，就是**阶级力量的对比和变化**。

当前的任务不是"设计完美的制度"，而是：
1. **帮助数字无产阶级认识到自己的阶级地位**（从"我是一个骑手"到"我是一个被算法剥削的劳动者"）
2. **把分散的、原子化的数字劳动者组织起来**（组织度从★提升到★★★）
3. **在斗争中逐步扩大变革力量的影响力**（等待力量的转化）

> **"革命不是请客吃饭，不是做文章，不是绘画绣花，不能那样雅致，那样从容不迫，文质彬彬，那样温良恭让。"**
> ——毛泽东《湖南农民运动考察报告》

制度设计是必要的——它为我们提供了目标方向。但通向目标的道路，不是"设计→实施"的工程师思维，而是"组织→斗争→在斗争中调整目标"的政治思维。

**这套方案，加上07号、09号的制度设计，再加上正确的组织方法，才构成完整的AI时代变革纲领。**
