# AI時代における各階級の分析

> 知識蒸留元：『中国社会各階級の分析』（1925年）
> 作成日：2026年5月2日
> 理論的出典：毛沢東『中国社会各階級の分析』『実践論』『矛盾論』、マルクス『資本論』

---

## 一、核心的論題

> **「誰が我々の敵か？誰が我々の味方か？この問題は革命の首要的問題である。」**
> ——毛沢東『中国社会各階級の分析』（1925年）

AI時代の変革の力はすでに形成されつつあるが、明確な**階級分析の枠組み**を欠いている。この問題を解決しなければ、あらゆる制度設計は的を射ないものとなる。

既存の07号および09号方案は一連の制度を設計しているが、決して次の問いに答えていない。**誰がこの変革を推進するのか？誰がこの変革に反対するのか？誰が争取可能な中間勢力なのか？**

本稿は、この空白を直接的に補完するものである。

## 二、AI時代の階級全景図

「経済的地位→政治的態度→行動傾向」という分析方法に従い、AI時代の社会集団を以下の五つに分類する。

```

│                    階級勢力スペクトル                        │
├─────────────────────────────────────────────────────────────┤
│                                                             │

│                │                    │                         │

│  ② 軍事AI複合体│  ④ 伝統資本階層    │  ⑥ 周辺化され淘汰される者 │

```

## 三、敵：変革への抵抗勢力

### 第一類：AI独占資本

**含む：** 大規模AIプラットフォームの所有者、株主、核心上級経営層（OpenAIのSam Altman、GoogleのPichai、MetaのZuckerberg、バイトダンスの張一鳴など、およびその背後にあるベンチャーキャピタル）

**経済的地位：** AI時代の最も中核的な三つの生産手段——データ（ユーザーの行動データプール）、計算能力（GPUクラスターおよびデータセンター）、アルゴリズム（基盤モデルおよび推薦システム）——を掌握している。

**世界年間収入規模（上位5大AIプラットフォーム）：** 1.5兆米ドル超。これは中程度の発展途上国のGDPに相当する。

**階級的特徴：**
- 産業時代のトラストよりも独占の度合いが高い——ネットワーク効果＋データフライホイール＋エコロジカルロックインが「勝者総取り」を形成
- 分配権を掌握——誰が見られるか（推薦アルゴリズム）、誰が信頼されるか（信用スコア）、誰が雇用されるか（AI面接）を決定
- インフラを掌握——チップ（NVIDIA）からクラウド（AWS/Azure/GCP）からモデル（GPT/Claude/Gemini）に至るまで

**政治的態度：** データ公有制、アルゴリズムの透明化、AI再分配に積極的に反対。ロビー活動、学術への資金提供、メディア支配を利用して「AIリスクには企業の自主規制が必要」という世論を形成する。

**戦略：**
- 彼らは主要な矛盾側であるが、現時点での力関係は圧倒的に不利
- 直接対決は失敗する——彼らの内部矛盾（市場競争、技術路線の相違）を利用する必要がある
- レーニンは「一つの帝国主義の内部にも矛盾がある」と述べた——NVIDIAとその顧客（OpenAI、Google）の間には利害の対立が存在する

**典型的人物像：**
```
Sam Altman（OpenAI CEO）：
  公には「AIは多くの仕事を破壊する」と言う → しかしUBIの財源としてAI税に反対
  → 「AIの利益は広く分配されるべき」と主張 → しかし企業評価額3000億ドルをデータ提供者には分配しない
  → これは「慈善の仮面をかぶった資本の本質」である

  **評価：** 個人の品德の問題ではない。構造的な位置が、彼に株主利益を守ることを強制しているのである
```

### 第二類：軍事AI複合体

**含む：** 軍事用AI請負企業（Palantir、Anduril、Raytheon、Lockheed Martin）、軍事AI研究開発機関、戦争予算に依存する研究システム

**経済的地位：** 世界の軍事AI市場は2030年までに3000億ドルに達すると予測される。これらの企業の利益率（30-50%）は民生用AI企業を大幅に上回る。

**階級的特徴：**
- ビジネスモデルが紛争と戦争に依存——平和はその利益に反する
- 政府と深く結合——「回転ドア」メカニズム（元国防長官→AI企業CEO→政府に戻りAI政策を策定）
- 技術倫理の制約が最小——軍事AIには「ユーザーの同意」問題が存在しない

**政治的態度：** AIの武器化規制、アルゴリズムの透明性要求、世界的なAIガバナンス条約に断固反対

**戦略：**
- 道義的に完全に孤立させる（しかし彼らの立場転換は期待できない）
- 彼らが政策と世論を操作する手段を暴露する
- 重要局面：**致命的自律兵器システム（LAWS）**の全面配備を阻止できるかどうか

**規模推計：** 世界で約500〜800社の核心企業、従業員約300万人（サプライチェーン含む）。

## 四、中間派：争取可能な勢力

### 第三類：技術中産階級 ——*変革の鍵を握る変数*

これは最も複雑で、かつ最も重要な集団である。細分化が必要である。

#### 3.1 上位技術エリート（アルゴリズムエンジニア、AI研究者、プロダクトマネージャー）

**経済的地位：**
- 年収：シリコンバレー30〜80万ドル、中国50〜200万元、その他の地域10〜30万ドル
- 資本としての意味での人的資本を有する——市場における希少性が彼らに交渉力を与えている
- しかし**生産手段を所有していない**——コードは会社に属し、モデルは会社に属し、データは会社に属する

**政治的態度：** 揺れ動く中間派
- アルゴリズムエンジニアは「技術中立」を崇拝し、政治に興味を示さない
- しかし「誰のために働くのか」という問題で分化が生じている——軍需企業への就職を拒否するAI研究者が増加（GoogleのAI研究者がProject Mavenに抗議して辞職、OpenAIのセキュリティチームが集団離脱）
- もし大規模なレイオフ（AIによるAIプログラマーの代替）が発生すれば、態度は急激に左傾化する

**戦略：**
- 「技術は中立ではない」ことを教育する——フーコーの知識＝権力分析
- 彼らの職業倫理を喚起する——「あなたが書く一行一行のコードには社会的結果が伴う」
- 彼らと資本の間の矛盾を利用する——資本が彼ら自身をAIで代替することを決定した時、彼らは覚醒する

**典型的事例：**
```
2024〜2026年のトレンド：
  OpenAIはGPT-4で人間のコンテンツ審査員を代替（ケニア人労働者が解雇）
  GoogleはAIで広告販売チームを代替（3万人が失業）
  MetaはAIにコードを書かせる → プログラマーが自らの製品によって代替され始める
  ますます多くのAI企業がAIにAIの訓練データをラベリングさせる → 人間のデータラベラーが排除される

トレンド：技術中産階級の「代替の連鎖」における位置が絶えず前進している
  → 昨日までは工場労働者が代替されていた
  → 今日はデータラベラーとカスタマーサービスが代替されている
  → 明日はプログラマー自身が代替されるかもしれない

**これこそが、彼らが進歩陣営へと向かう物質的基盤である**
```

#### 3.2 中層技術管理者（テクニカルディレクター、プロダクトVP、CTO）

**経済的地位：** ブルジョアジーに近い——ストックオプション、経営権力、意思決定への参加権を有する

**政治的態度：** 保守的であり、現行体制を維持する傾向がある

**戦略：** 主要な争取対象ではない。彼らに過剰な資源を費やすべきではない。

#### 3.3 周辺技術従事者（ITサポート、データアナリスト、初級開発者）

**経済的地位：**
- 年収：先進地域5〜15万ドル、グローバルサウス2〜5万ドル
- AIによる雇用安定性のリスクが極めて高い——大量の反復的コーディング、データ処理業務が代替される
- 技術エリートのような代替不可能性を欠く

**政治的態度：** 潜在的に急進的な力——彼らは技術中産階級の中でAIに代替される可能性が最も高く、変革の主張を受け入れる可能性が最も高い集団である

**戦略：** 重点的発展対象。彼らは技術中産階級とデジタルプロレタリアートとの間の接続橋梁である。

### 第四類：伝統資本階層

**含む：** AI産業以外の伝統的資本家——製造業、小売業、不動産業、伝統的金融

**経済的地位：** AI資本によって侵食され圧迫されつつある。小売はAmazonに破壊され、金融はアルゴリズム取引に打撃を受け、製造業はAIオートメーションによって労働者が代替されている。

**政治的態度：** 複雑かつ矛盾的
- 彼らとAI独占資本の間には**競争関係**が存在する——これは利用可能な矛盾である
- しかし彼らと労働者の間には依然として**搾取関係**が存在する——彼らを信頼することはできない
- 特定の議題においては「一時的な同道者」となり得る——例えばデータプライバシー保護を支持する（彼らのデータもAIプラットフォームに収集されているため）

**戦略：**
- 労働者の根本的利益を損なわない限りにおいて、彼らとAI資本の矛盾を利用することは可能
- しかし彼らを信頼できる同盟軍と見なしてはならない——自分たちの搾取的利害が脅かされれば、即座に裏切る
- 具体戦略：「反データ独占」の議題において限定的な戦術的連合を構築する

**毛沢東原典対照：**

> 「あの動揺し定まらぬ中産階級は、その右翼は我々の敵となり得、その左翼は我々の味方となり得る——しかし我々は常に彼らに警戒し、彼らに我々の戦線を乱させることのないようにしなければならない。」

——これはAI時代の技術中産階級と完全に符合する。

## 五、味方：変革の基本勢力

### 第五類：デジタルプロレタリアート ——*変革の主力軍*

#### 5.1 ギグワーカー（配達員、ライドシェア運転手、フードデリバリー要員）

**世界規模：** 約5〜8億人（パートタイム含む）、最も急速に成長している労働集団

**経済的地位：**
- 固定的雇用関係なし——法律上は「独立請負人」だが、実際にはアルゴリズムに完全に統制されている
- 労働保障なし——健康保険、年金、有給休暇がない
- 収入水準：最低賃金を下回る事例が普遍的に存在（プラットフォームの手数料控除後）
- 交渉力なし——評価が1ポイント下がれば、配車は半減する

**階級的特徴：**
- 最も直接的かつ顕著にアルゴリズムによって搾取されている——配車のたびに、経路のたびに、待ち時間の一分一分がアルゴリズムによって極限まで最適化される
- 原子化——配達員同士の繋がりはなく、互いに競争させられ、プラットフォームはアルゴリズムで「レース効果」を作り出す
- しかし組織化が始まっている——中国のフードデリバリー配達員のストライキ、米国Uber運転手の労働組合、英国Deliverooの集団訴訟

**政治的態度：** 現状に極度の不満を抱いているが、政治的方向性を必要としている

**戦略：**
- これは最も革命的な力である——彼らは最も搾取されている集団であり、失うものは何もない
- 彼らの組織化を支援する——プラットフォームの原子化戦略を突破する
- 彼らを「利益闘争」（単価引き上げ）から「制度闘争」（アルゴリズムの公正さ、データ権利）へと導く
- 典型的な突破口：**「アルゴリズム・ストライキ」**——集団ログアウト、集団低評価、集団での偽データ投入

#### 5.2 デジタル肉体労働者（データラベラー、コンテンツ審査員、AIトレーナー）

**世界規模：** 約5000万〜1億人

**経済的地位：**
- 国際分業の最底辺——ラベラーはケニアで時給2ドル、フィリピンで1.5ドル
- 心理的トラウマが深刻——コンテンツ審査員は暴力、児童虐待、テロリズムのコンテンツを継続的に視聴し、PTSD発症率が極めて高い
- 完全に不可視——ユーザーは「ChatGPTを訓練しているのはアルゴリズムではなく、ケニアの低賃金労働者である」ことを知らない

**階級的特徴：**
- 最も隠蔽された被搾取集団——彼らの労働は「見えない」（エレイン・エルスワースの言う「不可視労働」）
- ブランド認知なし——誰も彼らがどのモデルのために働いたかを知らない
- 代替可能性が最も高い——AIが人間のラベラーを代替している（AIでAIをラベリングする）、これは彼らにとって二重の抑圧である

**政治的態度：** 最も絶望的な集団——現時点では集団行動のための組織的条件を欠いている

**戦略：**
- 長期目標：「デジタルスウェットショップ」の廃絶
- 中期目標：透明化——AI企業に対しラベラーの労働条件と報酬の公開を要求
- 短期目標：彼らとAI独占資本の矛盾は不可調和である——問題は彼らにまだ発言のチャンネルがないことである

### 第六類：周辺化され淘汰される者

#### 6.1 AI代替失業者

**含む：** AIに代替された伝統産業の労働者——翻訳者、カスタマーサービス、初級会計士、初級弁護士、イラストレーター、データ入力担当者……

**規模とトレンド：**
- 2023〜2030年に世界でAIに直接代替されると予測される雇用：3〜5億件（Goldman Sachs、McKinseyなど複数機関の予測）
- しかも代替速度は過去の産業革命をはるかに上回る——「一世代の時間」ではなく「十年以内」

**経済的地位：** 旧来の生産手段（伝統的職業技能）を失い、かといって新しい生産手段（AIツールと計算資源）も獲得していない

**政治的態度：** 怒りを持っているが方向性は不明確——ポピュリズムや排外主義的感情に惹かれやすい

**戦略：**
- これは最も重要で、かつ最も危険な集団である
- もし彼らの怒りを正しい方向（AI独占資本への反対）に導かなければ、誤った方向（反移民、反グローバリゼーション、人種主義）に導かれる
- 正しい方向：**「AIがあなたの仕事を奪ったのではない。資本がAIを利用してあなたの仕事を奪ったのである」**

**歴史的参照：**
```
産業革命期：
  イギリスのラッダイトが機械を打ち壊した → 機械は確かに彼らの生計を破壊した
  しかし組織なき機械破壊は → 弾圧をもたらすだけで、変革をもたらさなかった
  最終的に組織化された労働組合が → 8時間労働制と労働保障を勝ち取った

AI時代：
  「AIを打ち壊す」ことはできない（技術それ自体は敵ではない）
  「AIの所有権を打ち壊す」べきである（誰がAIを支配するかが核心問題である）

  ラッダイトの教訓：
    彼らが打ち壊したのは機械ではなく、資本主義制度の「物質的現れ」であった
    しかし彼らは「機械」と「機械の資本主義的応用」を区別しなかった
  → 我々は第六類の集団が「AI技術」と「AIの資本主義的応用」を区別できるよう支援しなければならない
```

#### 6.2 デジタル難民

**含む：** デジタルアクセス能力を欠く人々——世界には依然として26億人がインターネットにアクセスできず（ITUデータ）、高齢者、低所得地域の人口

**戦略：** 彼らを「受動的被害者」と見なしてはならない——彼らは変革の障害ではなく、変革の受益者である。真に包摂的なAI社会は、彼らのニーズから出発しなければならず、「先にデジタルユーザーに奉仕し、その後で彼らを考える」というものではありえない。

**鍵となる判断：**

現在の力関係では、**変革勢力は保守勢力よりもはるかに弱い**。AI独占資本はメディア、計算能力、データ、モデル——ほとんど全てを掌握している。しかしこれは我々が何もすべきでないということを意味しない——むしろ逆に、『持久戦論』は我々に次のことを教えている。力関係は**正しい戦略**によって変えることができるのである。

## 七、階級分析の戦略的応用

### 7.1 主要な敵は誰か？

**当面の主要な敵 = AI独占資本**

「あらゆる資本家」でもなければ、「AI技術そのもの」でもない。これを明確にすることは極めて重要である——もし目標を大きすぎる（「資本主義の打倒」）に設定すれば、現在の力関係では全く実現可能性がない。もし目標を小さすぎる（「AIをより公正にする」）に設定すれば、資本に吸収されて改良主義に堕してしまう。

正しい中期目標：**データ公有制**——これは資本主義が受け入れられる限界（公共図書館、公教育に類似）であり、より深遠な変革への出発点でもある。

### 7.2 主要な同盟軍は誰か？

**デジタルプロレタリアート（ギグワーカー＋デジタル肉体労働者）が基本勢力である。**
**技術中産階級の下層が争取すべき主要な同盟軍である。**
**AI代替失業者は必ず争取すべき鍵となる大衆である。**

### 7.3 利用可能な矛盾の対象は誰か？

**AI独占資本 vs 伝統資本**——「データ独占」の議題において共通の利益が存在する
**AI独占資本内部**——大プラットフォーム vs 小企業、モデル企業 vs 計算資源供給者（NVIDIA）
**AI独占資本 vs 国家主権**——特にラテンアメリカ、ヨーロッパにおいて、国家は米国のAI覇権に対抗する動機を持つ

### 7.4 当面の行動方針

以上の分析に基づき、当面の行動方針は次の通りである：

```
戦略的方向：データ公有化（資本の廃絶ではなく、資本の限界を画定すること）
主要矛盾：AI独占資本 vs 全社会（伝統資本、国家主権を含む）
主要勢力：デジタルプロレタリアート + 技術中産階級下層
統一戦線：伝統資本中の反独占勢力 + 主権国家政府 + グローバルサウス
闘争形態：立法闘争（アルゴリズム透明化法）+ 経済闘争（データ配当）+ 組織闘争（デジタル労働組合）
```

## 九、総括：階級分析の視座からAIガバナンスを観る

階級分析を欠いたAIガバナンス方案は、地図のない航海のようなものである。

07号および09号方案は美しい制度の青写真を描いている——しかし**制度は自然に出現するものではない**。制度は階級勢力比較の産物である。データ公有制は「論理的に正しいから」という理由だけで自動的に実現するわけではない——それを推進する階級（デジタルプロレタリアート）が必要なのである。

**本稿の核心的結論：**

> AI時代の変革は、単なる「技術ガバナンス」の問題でもなければ、「倫理設計」の問題でもない——それはまず**政治問題**である。そして政治問題の核心とは、**階級勢力の比較とその変化**に他ならない。

当面の任務は「完全な制度を設計すること」ではなく、以下のことである：
1. **デジタルプロレタリアートが自らの階級的地位を認識するのを助けること**（「私は一人の配達員である」から「私はアルゴリズムに搾取される労働者である」へ）
2. **分散し原子化されたデジタル労働者を組織化すること**（組織度を★から★★★へ引き上げる）
3. **闘争の中で変革勢力の影響力を徐々に拡大すること**（力の転換を待つ）

> **「革命はご馳走を食べることでも、文章を書くことでも、絵を描いたり刺繍をすることでもない。そんなに優雅に、悠然と、上品に、そして温良恭譲であることはできない。」**
> ——毛沢東『湖南農民運動考察報告』

制度設計は必要である——それは我々に目標の方向性を提供する。しかし目標への道は、「設計→実施」というエンジニア的思考ではなく、「組織化→闘争→闘争の中で目標を調整する」という政治的思考によって切り開かれる。

**本方策は、07号、09号の制度設計と、正しい組織方法とを合わせることによって、初めて完全なAI時代変革の綱領となるのである。**
