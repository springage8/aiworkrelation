# AI劳动社会设计：马克思主义与社会学理论的知识蒸馏

## 项目说明

> 设计日期：2026年5月1日·五一国际劳动节
> 知识蒸馏系列：用经典理论照亮AI时代的劳动问题

### 缘起

AI技术正在以前所未有的速度重塑人类的劳动方式、生产关系和权力结构。然而，关于AI的讨论往往局限于技术效率和市场逻辑，**缺乏对深层社会关系变革的理论洞察**。

本项目从马克思主义出发，融合社会学经典理论，对AI时代的劳动问题进行一次系统性的**知识蒸馏**——将复杂的理论体系转化为精炼、准确、结构化的分析模块，最终形成对未来AI劳动社会生产关系的一份设计方案。

### 如何使用

1. 如果你想**理解AI时代的权力结构** → 从01（数据所有制）和05（福柯）开始
2. 如果你想**理解AI对劳动者的影响** → 从02（异化）和04（韦伯）开始
3. 如果你想**寻找变革方案** → 直接跳到07（综合设计方案）
4. 如果你需要**快速对照概念** → 参考08（术语表与索引）

### 关键洞察

- **数据是AI时代的生产资料**——它的社会化和私人占有之间的矛盾，是当代资本主义的核心矛盾
- **算法管理是新型的异化形式**——四个维度的异化在数字时代深化并转化
- **AI没有推翻劳动价值论**——它暴露了资本主义价值核算体系的深层矛盾
- **解决之道不在技术本身**——而在于生产关系的变革：数据公有制、算法民主制、自由劳动
