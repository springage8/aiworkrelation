# El Camino de Jinggangshan hacia la Propiedad Común de Datos — Estrategia de Bases

> Destilado de *¿Por Qué Puede Existir el Poder Político Rojo en China?* (1928), Mao Zedong
> Fecha: 2 de mayo de 2026

---

## Tesis Central

> **"La existencia prolongada de una o varias pequeñas áreas de poder político rojo rodeadas por un régimen blanco—esto es algo nunca antes visto en el mundo."**

La transformación en la era de la IA enfrenta el mismo problema estructural: ~100% de la infraestructura global de IA está controlada por el capital monopólico.

La respuesta: **Construir bases de apoyo.** No "derrocar el viejo sistema primero", sino construir "fortalezas" del nuevo sistema dentro de las **grietas** del viejo.

## Las Tres Candidatas a Base

### Candidata 1: Fideicomiso de Datos de Salud Pública en América Latina ★RECOMENDADA

**Lugar:** Chile o Brasil

**Por qué aquí:**
- Voluntad política de "soberanía digital" más fuerte del mundo
- Salud pública: "baja sensibilidad, alta necesidad"
- Población adecuada para piloto (Chile 19 millones)
- Potencial de efecto demostración para todo el Sur Global

**Plan:**
```
Fideicomiso Público de Datos de Salud:

Datos: Hospitales públicos (anonimizados) + vigilancia epidemiológica + ciudadanos voluntarios
Usos: Modelos de diagnóstico localizados (dengue, Chagas, tuberculosis)
                      Optimización de recursos de salud
                      Investigación farmacéutica

Gobernanza: Consejo—gobierno 1/4 + hospitales 1/4 + ciudadanos 1/4 + expertos 1/4
Distribución: 50% al sistema de salud + 30% dividendo de datos + 20% costos operativos

Indicadores de éxito (2 años):
  → 50%+ de datos hospitalarios cubiertos
  → 1+ modelo de IA diagnóstico en uso (precisión >90%)
  → Participación voluntaria >30%
```

### Candidata 2: Cooperativa de Datos de Transporte Urbano en Europa

**Lugar:** Hamburgo (Alemania) o Utrecht (Países Bajos)

**Ventaja:** Marco legal más favorable (GDPR, Ley de Gobernanza de Datos de la UE)

### Candidata 3: Cooperativa de Datos de Repartidores en China

**Lugar:** Ciudad china de segundo nivel

**Ventaja:** Mayor escala de trabajadores de plataformas del mundo (solo Meituan: 7+ millones)

## Modelo Económico

### Autosostenibilidad financiera

| Etapa | Fuente de ingresos |
|-------|-------------------|
| Temprana (1-2 años) | Fondos públicos/filantrópicos (~$0.5-2M/año) |
| Media (2-5 años) | Licencias a usuarios comerciales (seguros, farmacéuticas) |
| Madura (5+ años) | Excedente distribuido como dividendo de datos |

### Fórmula de distribución

```
Ingreso total anual = Σ(licencias comerciales) + presupuesto público

Prioridad:
  1° Costos operativos (30%)
  2° Fondo de reserva (20%)
  3° Dividendo de datos (50%)
    → 60% base igualitaria + 40% según contribución
```

## Principios de Gobernanza

**Centralismo democrático aplicado a la propiedad común de datos:**

- **Democracia:** Una persona, un voto para elegir el consejo. Decisiones importantes pasan a votación general.
- **Centralismo:** El consejo gestiona las decisiones diarias. Una vez decidido, se ejecuta.

**Sin democracia = burocracia de datos. Sin centralismo = ineficiencia.**

## Camino de Expansión

### Horizontal: de 1 fideicomiso a 10

```
1. Documentar → kit de herramientas de código abierto
2. Capacitar → operadores del piloto inicial son "instructores semilla"
3. Conectar → alianza de fideicomisos de datos
4. Estandarizar → estándares nacionales basados en la práctica
```

### Vertical: de baja sensibilidad a alta

```
Fase 1: Datos públicos (gobierno, clima) → Fácil, construir confianza
Fase 2: Datos de salud, transporte, educación → Alto impacto, alta participación
Fase 3: Datos económicos (consumo, finanzas) → Toca intereses centrales del capital
Fase 4: Poder de gestión algorítmica → La transformación más profunda
```

## Errores a Evitar

1. **Esperar el plan perfecto** — Nunca llegará. Practicar es el criterio de la verdad.
2. **Apresurarse a expandir** — Sin base sólida, la expansión lleva al colapso.
3. **Trabajar a puerta cerrada** — El fideicomiso debe ser "proyecto del pueblo" desde el día uno.
4. **Determinismo tecnológico** — El núcleo es la confianza y la gobernanza, no la tecnología.
5. **Pesimismo** — "Una chispa puede iniciar un incendio en la pradera."
