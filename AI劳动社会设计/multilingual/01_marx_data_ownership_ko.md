## I. 중심 테제

**데이터는 생산수단으로서 사적 횡령과 사회화된 생산 사이의 모순을 발생시킨다——이것은 AI 시대 자본의 근본적 모순의 새로운 형태이다.**

---

## II. 추론 과정

### 2.1 '생산수단' 개념에서 출발하기

마르크스는 『자본론』 제1권에서 '생산수단'을 다음과 같이 정의한다:

> "노동수단과 노동대상의 총체" (『자본론』 제1권 제5장)

AI 시대에서:
- **노동수단(도구)** → 알고리즘 모델, 컴퓨팅 클러스터, 훈련 프레임워크
- **노동대상(원자재)** → 데이터(텍스트, 이미지, 행동 기록, 생체 정보)

**핵심 통찰:** 데이터는 전통적 의미의 '원자재'가 아니다——그것은 **집단적 인간 사회 활동의 부산물**이지, 개별 자본가의 투자 산물이 아니다.

### 2.2 AI 시대 '본원적 축적'의 재발 추적하기

마르크스는 『자본론』 제1권 제24장에서 '소위 본원적 축적'에 대해 서술한다:

> "그들의 수탈의 역사는 피와 불의 문자로 인류의 연대기에 기록되어 있다."

**AI 시대의 '데이터 본원적 축적'은 다음과 같이 나타난다:**

| 산업 시대 | AI 시대 |
|----------------|--------|
| 인클로저 운동——농민 토지 수탈 | 플랫폼이 사용자 데이터를 무보수로 수집 |
| 식민지 약탈 | 글로벌 사우스로부터의 저비용 주석 데이터 획득 |
| 노예 무역 | 은폐된 콘텐츠 모더레이션 노동(케냐, 필리핀 등) |
| 아동 노동 | 아동 데이터의 무차별 수집 |

**도출된 결론:** AI 기업의 '데이터 우위'는 시장 경쟁의 자연스러운 결과가 아니라, **제도적 수탈**(장황한 이용 약관, 강제적 승인, 데이터 블랙박스화)을 통해 달성된 본원적 축적이다.

### 2.3 '상품 물신주의'에서 '데이터 물신주의'로

마르크스는 『자본론』 제1권에서 상품 물신주의를 분석한다:

> "그것은 인간 상호 간의 일정한 사회적 관계에 불과하지만, 여기서 사람들에게는 사물과 사물 사이의 관계라는 환상적인 형태를 취한다."

**데이터 물신주의의 발현:**
- 데이터는 '객관적으로 중립적인' 자원으로 간주되어, **데이터 뒤에 숨은 인간 관계**(누가 생산하고, 누가 수집하며, 누가 이익을 얻는지)가 은폐된다
- 알고리즘 결정은 '과학적이고 공정하다'고 간주되어, **훈련 데이터에 내재된 계급적 편향과 역사적 불의**가 무시된다
- '데이터 중심'이 새로운 이데올로기가 되어, 마치 노동자의 참여 없이 데이터 자체가 가치를 창출할 수 있는 것처럼 여겨진다

### 2.4 데이터 자본의 특수한 순환

마르크스의 자본 일반 공식: **M → C → M'**

AI 자본의 특수한 순환:

```
데이터 수집 → 모델 훈련 → 애플리케이션 배포 → 더 많은 사용자 → 더 많은 데이터 → 더 강력한 모델 → 더 높은 이윤
   ↑                                                                                                        │
   └───────────────────── 데이터의 무상 환류 ────────────────────────────────────────────────────────────────────┘
```

**핵심 모순:** 데이터 생산에서의 **비용의 사회화**(사용자의 데이터 무상 생산, 오픈소스 커뮤니티의 코드 기여, 공공 지식의 스크래핑) 대 데이터로부터의 **이윤의 사적화**(주주 배당, 경영진 보상, IPO 현금화).

## III. 지식 증류: 핵심 논증 구조

### 논증 1: 데이터는 '의사 상품'이다

```
전통적 상품은 경합재이다(내 소비가 당신의 소비를 배제한다)
          ↓
데이터는 비경합재이다(복제 가능, 공유 가능, 한계 비용이 0에 수렴)
          ↓
그러나 자본은 기술적 장벽(API 제한), 법적 장벽(저작권, 특허),
플랫폼 락인 효과를 통해 인위적으로 경합성을 창출한다
          ↓
데이터의 '상품 형태'는 자본에 의해 강제로 부과된 것이지,
데이터의 자연적 속성이 아니다
```

### 논증 2: 사용자는 '이중으로 착취당하는' 존재이다

```
공장에서 노동자는 잉여 노동 시간을 위해 착취당한다
          ↓
디지털 플랫폼에서의 노동자:
(1) 소비자로서——데이터가 수집된다(무보수)
(2) 프로슈머로서——콘텐츠를 생산한다(트렌드 형성, 댓글 작성)하여 플랫폼에 트래픽을 창출한다
          ↓
이것은 '이중 무보수 노동'의 형태이다: 소비 행위 자체가 생산 행위이다
```

### 논증 3: AI 시대의 새로운 계급 구조

```
전통적 마르크스주의 이계급 모델 → AI 시대의 정교화:

부르주아지(플랫폼 소유자, 주주)
    ↓
경영진 + 테크노크라트 엘리트(알고리즘 엔지니어, 프로덕트 매니저——고용되었으나 착취당하는가?)
    ↓
디지털 중산층(자체 미디어 제작자, 지식 노동자——표면적으로는 독립적이나 실제로는 플랫폼에 의존)
    ↓
긱 워커(라이더, 운전기사, 주석 작업자——알고리즘에 의해 직접 통제됨)
    ↓
보이지 않는 노동자(콘텐츠 모더레이터, 데이터 주석 작업자——심리적 트라우마, 저임금, 보호 없음)
    ↓
배제된 사람들(알고리즘에 의해 배제되고 AI로 대체된 사람들——마르크스의 '산업 예비군'의 AI 버전)
```

## V. 흔한 오해와 정정

| 오해 | 정정 |
|------------------|------------|
| "데이터는 새로운 석유이다" | 석유는 고갈성이며 경합재이다. 데이터는 사용할수록 가치가 증가한다. 이 비유는 잘못되었다. |
| "사용자는 자발적으로 데이터 수집에 동의한다" | 대체 플랫폼이 없고 '동의하거나 떠나거나' 식의 이용 약관 하에서는 동의가 존재하지 않는다. |
| "AI가 가치를 창출한다" | AI 자체는 가치를 창출하지 않는다——그것은 무수한 노동자가 이전에 창출한 지식의 가치를 이전하고 집중시킬 뿐이다. |

## VI. 실천적 적용: 사례 연구

### 사례: Amazon 창고 알고리즘 관리

- **생산수단:** 창고, 재고 시스템, WMS 알고리즘——모두 Amazon이 통제
- **노동 과정:** 알고리즘이 피킹 경로, 시간을 설정하고, 효율을 모니터링하며, 자동 해고한다
- **데이터 추출:** 모든 노동자의 동작이 기록되어 알고리즘 최적화에 사용되지만, 노동자는 이 데이터에 접근할 수 없다
- **결과:** 노동자는 알고리즘의 '수동 액추에이터'로 전락하고, 인간의 주체성은 완전히 소멸된다

### 사례: ChatGPT의 훈련 데이터 출처

- **무보수 노동 평가:** 각 훈련 데이터가 0.01위안의 가치가 있다고 가정하면, GPT-4의 1.5조 토큰은 수조 위안에 달하는 암묵적 노동 가치에 해당한다
- **오픈소스 커뮤니티 노동의 착취:** GitHub 코드, Wikipedia, Reddit 토론——공공 지식의 자본화
- **데이터 주석 노동:** OpenAI는 케냐에서 주석 작업자들을 시간당 2달러 미만으로 고용하여 유해 콘텐츠를 처리하게 했다
