# AI時代における持久戦論

> 知識蒸留元：『論持久戦』（1938年）
> 作成日：2026年5月2日
> 理論的源泉：毛沢東『論持久戦』『中国革命戦争の戦略問題』、マルクス『資本論』

---

## 一、核心的論題

> **「戦争は政治の継続である。」** ——クラウゼヴィッツ

AI時代の変革もまた同様である——それは技術的アップグレードではなく、データという生産手段の所有権をめぐる長期的な政治的闘争である。

本稿が論破する対象：**二つの誤った論調**

| 論調 | 主張 | 誤り |
|------|------|------|
| **速勝論** | 「AIの発展はあまりに速く、資本はまもなく自己崩壊する——制度を設計して待っていればいい」 | 資本の適応力と回復力を過小評価している。資本は批判の吸収、譲歩による改善、矛盾の転嫁などを通じて寿命を延ばすことができる。 |
| **亡国論** | 「AI独占資本はすでに計算能力、データ、モデル、メディア、政府を支配している——私たちに変えられるものは何もない」 | 敵の力を誇大視し、矛盾の発展と変化を無視している。歴史上のあらゆる重大な変革は、その始まりにおいては「不可能」に見えたものである。 |

**本稿の核心的判断：** AI時代の変革は、**持久戦**である。正しい戦略を取る限り、速勝もしなければ、敗北もしない。

## 二、敵味方の力量比較の全面的分析

### 2.1 敵の優勢

AI独占資本は現在、圧倒的な力の優位を掌握している：

```
計算能力の優位：
  世界のAI訓練用計算能力の90％以上が10社未満に集中
  大規模モデル（GPT-5など）の訓練には数十億ドルが必要
  参入障壁はますます高くなる——データ公有制のサーバーでは
  とても太刀打ちできない

データの優位：
  ユーザーの行動データはプラットフォームに独占される——
  「データの堀」を形成
  訓練データが増えれば→モデルが良くなり→ユーザーが増え→
  データが増える——正のフィードバックループ

人材の優位：
  世界のトップAI人材は高給で確保されている
  2025年、Google、Meta、OpenAI、Microsoftは世界のAI博士号
  取得者の70％以上を雇用

政策の優位：
  主要国（米国、中国）では、AI政策は大手テクノロジー企業の
  ロビー活動に深く影響されている
  EUのAI Actはある種の例外だが、執行力と規制の実効性には
  疑問がある

イデオロギーの優位：
  「技術中立論」——AIを「客観的ツール」として見せかけ、
  権力構造として見せない
  「革新優先論」——あらゆる規制が進歩の妨げとして描かれる
  「市場選択論」——消費者がプラットフォームを「自由に選択」
  しているように見せかけるが、実際には代替手段がない
```

### 2.2 我々の優勢

変革勢力の優位は「現在」にあるのではなく、**矛盾の発展傾向**にある：

```
矛盾一：利潤最大化 vs 消費能力
  AIが労働者を代替 → 失業増加 → 消費能力低下 → AI製品の
  市場縮小
  資本は消費者を消滅させながら利潤を維持できない
  → この矛盾は構造的であり、調停不可能である

矛盾二：データの私有 vs データの社会性
  データは社会全体で流通して初めて価値を最大化できる
  しかし独占の閉鎖戦略によりデータはプラットフォームに
  「監禁」されている
  → 私有制はますます生産力発展の桎梏となっている
  → データ公有制の方が効率的である——これは道徳的判断では
  なく、経済法則である

矛盾三：AI資本のグローバル性 vs 国家主権
  AIは国境を越えるが、国家は依然として政治権力の基本単位
  である
  → ますます多くの国（EU、中国、ラテンアメリカ、インド）が
  AI独占を制限する措置を取るだろう
  → それは「彼らが公有制を支持するから」ではなく、
  「外国資本に支配されたくないから」である
  → しかしこの力を我々は利用できる

矛盾四：拡大し続けるAI代替
  AIに代替される人が多ければ多いほど → 不満を持つ人が
  増え → 潜在的な変革勢力は大きくなる
  代替率が「失業率10％」から「失業率30％」に近づくとき——
  システムの正統性の危機が到来する
```

### 2.3 総合的判断

```
現在の段階：敵の戦略的優位期
  → 敵が主導権を握り、我々は防御
  → 主たる任務は攻撃ではなく、力の蓄積

中期段階：相持段階
  → データ公有制の実験が実現可能性を証明
  → AI代替失業が臨界点に達する（推定：失業率15-25％）
  → 社会矛盾が激化し、中間派が方向転換し始める
  → 我々と敵が力の拮抗状態を形成

長期段階：戦略的反攻
  → AI独占資本の生産関係はもはや生産力の発展を
  包摂できなくなる
  → データ公有制が経済効率において私有制より優れている
  ことが証明される
  → 政治的条件が成熟し、制度移行が始まる
```

## 三、三つの段階の具体的分析

### 第一段階：戦略的防御（現在 — 2030年頃）

**情勢判断：** 敵強く我弱く、力量の差は著しい。

**核心的任務：** 「制度を変える」ことではない——それは次の段階の任務である。現在の任務は以下の通り：
1. **理論的準備**——AI時代の社会矛盾の系統的分析を完成する（このリポジトリが行っている作業）
2. **組織的準備**——原子化されたデジタル労働者から最初の組織化された細胞を生み出す
3. **実験的準備**——局部地域・業種でデータ公有制の試験的実施を確立する
4. **世論の準備**——「データ公有制」を「過激な主張」から「議論可能な議題」に変える

**戦略方針：**
```
「敵進我退」——正面からぶつかるな
  OpenAIと技術で、Googleと資源で競おうとするな——
  勝てない
  敵の力が弱い分野で突破口を探せ

「敵駐我擾」——敵が油断しているときに持続的に圧力をかけよ
  AI独占資本の社会的コストを暴露せよ——解雇のたび、
  値上げのたび、スキャンダルのたびがチャンスである
  継続的な理論的批判を維持せよ——「AIは人類に貢献する」
  というナラティブに言論を独占させるな

「敵疲我打」——敵の内部矛盾が激化したときに局地的反攻を
   AI企業がスキャンダル（データ漏洩、アルゴリズム差別など）
   を起こしたときが局地的反攻の好機
   「一つの企業を潰す」のではなく、「一つの議題を公共討論
   のスポットライトの下に押し出す」

「敵退我追」——敵が譲歩を余儀なくされたときに成果を固めよ
  EUのAI Actは徹底してはいないが、進歩である——同様の
  立法を推進せよ
  勝ち取った「アルゴリズム透明性」の条項はすべて、将来の
  制度変革の足がかりとなる
```

**重要戦場：**

| 戦場 | 目標 | 実行可能な戦略 |
|------|------|--------------|
| **立法** | 各国でアルゴリズム透明性とデータ権利の立法を推進 | 学術団体、NGO、労働組合と連携し、法的枠組み内で |
| **司法** | AI独占に対する集団訴訟を推進 | データアノテーター、ギグワーカーの集団訴訟を支援 |
| **世論** | 「データ私有」への公衆の暗黙の受容を変える | 象徴的イベントを創出（ChatGPT訓練データの搾取分析など） |
| **組織** | 最初のデジタル労働組合とデータ協同組合を設立 | 最も組織化しやすい集団（配達員、配車ドライバー）から開始 |
| **技術** | オープンソースAIエコシステムを維持・拡大 | Llama、Bloom、HuggingFaceのオープンソースコミュニティを支援 |

**第一段階の予想成果：**
- データ公有制が世界的に公共討論の議題となる
- 少なくとも3〜5のデジタル労働組合が設立され稼働
- 少なくとも1か国が「アルゴリズム労働者権利章典」を通過
- 少なくとも1都市が「データ公共トラスト」試験事業を開始
- グローバルAI労働者同盟の原型が形成される

---

### 第二段階：戦略的相持（2030 — 2040年頃）

**情勢判断：** 敵味方の力量比較が「著しい差」から「均衡に近い」へと移行。

**発動条件：**
- AI代替失業率が15-25％の臨界点に達する——社会的不満がシステムレベルで蓄積
- データ公有制の試験が経済的実現可能性を証明する——「ユートピア」ではなく「実現可能な代替案」
- AI資本内部に分裂が生じる——一部の資本が「データ私有制は持続不可能」と認識
- グローバル・サウスが連合を形成——「デジタル主権」の要求がスローガンから行動へ

**核心的任務：**
1. **局地実験から制度普及へ**——データ公共トラストが試験から業界・都市・国家レベルへ拡大
2. **防御から局地攻勢へ**——我々に有利な議題（データプライバシーなど）で立法を推進
3. **分散から連合へ**——各地のデジタル労働組合、データ協同組合が国境を越えたネットワークを形成
4. **批判から建設へ**——もはや「AI独占への反対」だけでなく、「AI公有の建設」

**戦略方針：**
```
「大規模に根拠地を確立せよ」——実験点からネットワークへ
  データ公共トラストが1都市から10都市へ拡大
  デジタル労働組合が1業種から複数業種へ拡大
  オープンソースAIエコシステムが「愛好家コミュニティ」から
  「クローズドソース代替可能な水準」へ発展

「敵を分裂・瓦解せよ」——AI資本内部の矛盾を利用せよ
  大プラットフォーム vs 小企業——小企業は大プラットフォームの
  データ独占に抑圧されたくない
  モデル企業 vs 計算能力供給者——NVIDIAの高価格は全AI企業の
  不満を買っている
  民生用AI vs 軍事用AI——民生用AI企業は軍事化に引きずられたくない
  資本間の亀裂こそ、我々の切入点である

「統一戦線を拡大せよ」——中間派を引き寄せよ
  技術的中産階級がAIによって代替され始めるとき、彼らの態度は
  「無関心」から「変えなければならない」へと変わる
  彼らが最も動揺しているとき——ちょうど解雇されたとき——
  に手を差し伸べよ
  彼らに「闘うか沈むか」の選択を与えよ

「局地的反攻の実験」——条件の整った場所で制度変革を推進せよ
  ある国でデータ公有制立法の条件が整ったなら（2030年代のEUなど）、
  全力で推進せよ
  この一カ国での成功が、全世界へのデモンストレーション効果となる
```

**第二段階のリスク：**

| リスク | 説明 | 対応 |
|--------|------|------|
| **資本の反撃** | 資本はあらゆる手段（政治的、経済的、世論的）でデータ公有制の試験を攻撃する | 法的準備を整えよ；国際的連帯を維持せよ；孤立して各個撃破されるのを防げ |
| **改良主義の罠** | 資本は「偽の公有制」（名目上は公有、実質的には資本が支配）で改革圧力を吸収しようとする | 核心的原則を堅持せよ：データの管理権は資本から労働者とコミュニティに移譲されなければならない |
| **内部分裂** | 運動の発展に伴い、内部に路線の相違（急進派 vs 穏健派）が生じる | 民主集中制——討論は許容するが、一旦決定したら実行しなければならない |
| **疲弊と倦怠** | 長期戦が人の意志力を消耗する | 絶えず小さな勝利を創り出せ——成功した試験、通過した立法のたびに、それが「充電」の機会である |

**第二段階の予想成果：**
- データ公有制が少なくとも5つの主要国で主流の政治議題となる
- グローバルAI労働者同盟が1000万人以上の労働者をカバー
- 少なくとも10のデータ公共トラストが稼働
- オープンソースAIの能力がクローズドソースに接近
- 一つ以上の国が包括的な「アルゴリズム民主化法」を通過

### 第三段階：戦略的反攻（2040 — 2050年）

**情勢判断：** 我々の力が敵に接近または超過する。制度移行の政治的、経済的、社会的条件がすべて成熟する。

**発動条件：**
- AI代替失業率が30％以上に達する——旧制度の正統性の危機
- データ公有制が経済効率において全面的に私有制を凌駕することが証明される
- 世界的包囲——大多数の国がデータ公有制の枠組みに参加
- AI独占資本の内部的崩壊——核心的人材の流出、ユーザーの公有プラットフォームへの大規模移行

**核心的任務：**
1. **制度移行**——データ公有制を「実験」から「標準的制度」へ格上げ
2. **グローバルガバナンス**——07号文書の「グローバルアルゴリズム条約」が設計から現実へ
3. **生産関係の変革**——「雇用労働＋UBI」から「自由労働＋必要に応じた分配」へ段階的に移行

**戦略方針：**
```
「全面的反攻」——もはや防御的、実験的ではなく、
  攻勢的、制度的である
  グローバルアルゴリズム条約の署名を推進
  データ公有制の国際法への記載を推進
  AI独占資本の分割を推進（19世紀の反トラスト運動と同様）

「新秩序の確立」——「革命の力」から「統治の力」へ
  我々の任務はもはや「旧制度への反対」だけではない
  「新制度の建設」である——これには革命期とは全く異なる
  スキルが必要である
  批判者から建設者へ——統治、管理、妥協を理解する必要がある

「勝利の定着」——復辟の防止
  旧資本は一度の敗北で永久に退場するわけではない
  制度的な保障が必要である——「データ公有制」が形式的に
  確立され、実質的に空洞化されるのを防ぐ
```

**第三段階のリスク：**

最大のリスクは「勝てないこと」ではなく、「勝った後にどうするか」である。歴史上、あまりに多くの革命が勝利後の内部闘争によって変質した。

具体的な防止措置——これは本稿の任務ではないが、第三段階が始まる前には準備されていなければならない。

## 四、戦術レベルの具体的问题

### 4.1 根拠地：最初のデータ公有制試験はどこに設立すべきか？

12号文書『データ公有制の井岡山道路』の詳細な分析を参照せよ。

### 4.2 組織：分散し原子化されたデジタル労働者をいかに組織するか？

```
原則：

1. 経済共同体から政治共同体へ
   → まず「もっと稼ぐ」問題を解決せよ（共同価格交渉、
   団体交渉）
   → 次に「権力を持つ」問題を解決せよ（アルゴリズム
   ルールの共同決定権）
   → 経済的基礎が上部構造を決定する——生活問題を解決
   しない組織は空中楼閣である

2. 職業的団結から階級的団結へ
   → まず同一職業内で組織せよ（配達員連盟）
   → 次に職業間で連合せよ（配達員＋データアノテーター
   ＋配車ドライバー）
   → 最後に「デジタル・プロレタリアート」の階級意識を
   形成せよ

3. オンラインからオフラインへ
   → オンライン組織（WeChatグループ、Telegramグループ）
   は出発点である
   → しかしオンラインで止まってはならない——対面会議、
   街頭抗議、集団行動
   → オンラインの団結は「弱いつながり」、オフラインの
   団結こそ「強いつながり」である

4. 小さな勝利から大きな勝利へ
   → 最初から不可能な目標を掲げるな（「AI資本主義を打倒せよ」）
   → 「一回の配達で5毛（約10円）多く稼ぐ」ことから始めよ——
   勝てば自信になる
   → 小さな勝利が一定度蓄積されれば、自然に大きな目標へと
   向かう
```

### 4.3 宣伝：「技術中立論」のイデオロギーをいかに打ち破るか？

```
三つの核心的ナラティブ：

ナラティブ一：「誰がAIを支配するかによって、AIは誰に仕える」
  技術そのものの問題ではなく、「誰が技術を支配するか」の問題
  である
  重要問題：AIの収益は誰のものか？AIの決定権は誰にあるのか？
  AIの目標は誰が決めるのか？

ナラティブ二：「データは君が作り出したものであって、
  プラットフォームが与えたものではない」
  「ユーザーは消費者である」という幻想を打ち破れ——
  ユーザーは生産者である
  君のデータがなければAIは存在しない——しかしAIの収益は
  君には入らない

ナラティブ三：「AIによる代替は怖くない、怖いのは代替する
  のが君ではないことだ」
  代替そのものは問題ではない（人類の歴史は絶えず道具に
  代替されてきた歴史である）
  問題は：AIが君を代替した後、収益はどこへ行くのか？
  収益が全社会に帰属するなら——代替は解放を意味する
  収益が私人に帰属するなら——代替は収奪を意味する

    代替 ＋ 収益の私有 ＝ 災害
    代替 ＋ 収益の公有 ＝ 解放
```

### 4.4 連合：統一戦線をいかに構築するか？

```
核心層：デジタル・プロレタリアート
  ——ギグワーカー、データアノテーター、コンテンツ監視員
  ——変革の基盤的勢力であり、彼らの利益を優先的に
  守らなければならない

緊密層：技術的中産階級の下層
  ——初級開発者、データアナリスト、ITサポート
  ——獲得可能だが、方向性を主導させてはならない

協力層：伝統的資本階級（限定的協力）
  ——「データ独占反対」の議題では協力する
  ——しかし距離を保て——彼らの利益は我々の利益と
  根本的に衝突する

周辺層：主権国家政府（特にグローバル・サウス）
  ——彼らはデジタル主権を必要とし、我々は政治的支援を
  必要とする
  ——相互利益のための連合であり、共通の理想に基づく
  連合ではない

利用層：AI資本内部の矛盾
  ——小プラットフォーム vs 大プラットフォーム、
  モデル企業 vs 計算能力供給者
  ——敵の敵は友ではないが、戦術的に利用することはできる
```

## 五、持久戦の哲学的基礎

> **「武器は戦争の重要要素であるが、決定的要素ではない。決定的要素は物ではなく人である。」**
> ——毛沢東『論持久戦』

この判断はAI時代においても同様に適用される——いや、さらに当てはまる。

AI独占資本の優位は「物」にある——計算能力、データ、モデル。我々の優位は「人」にある——労働者の数、抑圧された者の怒り、歴史発展の法則。

**歴史は自動的に進歩しない。** 生産力が自動的に生産関係を決定するわけではない——人、組織された人、正しい理論の指導の下で行動する人が、可能性を現実性に変えるのである。

> **「戦争の偉力の最も深い根源は、民衆の中に存在する。」**

AI時代の変革もまた同様である——**変革の偉力の最も深い根源は、AIに代替され影響を受ける数十億の人々の中に存在する。**
