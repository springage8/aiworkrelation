# Prvomajski Manifest: Oblikovanje Človeških Proizvodnih Odnosov v Družbi UI Dela

**Datum:** 1. maj 2026 — Mednarodni Dan Delavcev
**Teoretični Viri:** Marx, Weber, Durkheim, Foucault, Bourdieu, Baudrillard, Žižek, Habermas, Ellul

## Temeljna Diagnoza: Temeljno Protislovje Dobe UI

> *"Rokov mlin vam daje družbo s fevdalnim gospodom; parni stroj vam daje družbo z industrijskim kapitalistom."* — Marx

Umetna inteligenca je največji preskok v proizvodnih silah v človeški zgodovini. Toda proizvodne sile same po sebi ne prinesejo avtomatično osvoboditve — **spremeniti se morajo tudi proizvodni odnosi**.

**Temeljno protislovje našega časa:**
- **Proizvodne sile so postale družbene** — podatki, algoritmi in računska moč so kolektivni produkt milijard ljudi
- **Proizvodni odnosi ostajajo zasebni** — iste vire monopolizira peščica korporacij
- **Rezultat:** UI, ki bi lahko osvobodila človeštvo pomanjkanja, namesto tega poglablja neenakost, nadzor in odtujitev

**Izbira pred nami:**
| **Pot A — UI Fevdalizem** | **Pot B — UI Emancipacija** |
|---|---|
| Nekaj jih nadzoruje algoritme, večina postane podatkovni tlačani | UI postane orodje za osvoboditev vsega človeštva |
| Vojna avtomatizirana z UI, viri umetno pomanjkljivi | Pomanjkanje odpravljeno, ustvarjena materialna osnova za mir |
| Algoritmično upravljanje poglablja vse štiri dimenzije odtujitve | Prosti čas postane pravo merilo bogastva |

---

## Trije Temeljni Principi

### Princip 1: Podatki Kot Javna Dobrina
| Zasebni podatki | → | Neodtujljiva osebna pravica (NI blago) |
| Družbeni podatki | → | Javni sklad (kot javna knjižnica) |
| Kritični algoritmi | → | Odprtokodne javne dobrine |
| Računska infrastruktura | → | Javna storitev (kot voda in elektrika) |

### Princip 2: Algoritmična Demokracija
- Pravila, ki jih določijo algoritmi → morajo biti predmet demokratičnega premisleka tistih, ki jih zadevajo
- Temeljne zahteve: **Razumljiv, Izpodbojen, Odgovoren**
- Cilji optimizacije: ne le učinkovitost, ampak pravičnost, dostojanstvo, blaginja

### Princip 3: Svobodno Delo
- Potreben delovni čas: zmanjšan na ~12–20 ur na teden
- Delo preneha biti zgolj sredstvo za preživetje → postane "prva življenjska potreba"
- Univerzalni temeljni dohodek (UBI) zagotavlja materialno življenje
- Prosti čas postane prostor za poln človeški razvoj

## Pet Stebrov Institucionalnega Oblikovanja

| Steber | Rešen Problem | Inovacija |
|--------|---------------|-----------|
| **Sistem Podatkovnih Dividend** | Platformni monopoli zajamejo 100% vrednosti podatkov | Javni Sklad za Podatke → razdeli 50% dobička UI nazaj vsem ljudem |
| **Transparentnost Algoritmov in Revizija** | Algoritmi so črne škatle brez legitimnosti | Tirinivojska revizija: notranja → neodvisna → družbena (javno sodelovanje) |
| **Digitalna Komuna** | Atomizacija dela, odtujitev | Organizacije UI-sodelovanja v lasti delavcev, demokratično vodene |
| **Časovna Suverenost** | Kapital kolonizira vse budne ure | Največ 20-urni delovni teden, plačan kreativni dopust, pravice "brez povezave nesleden" |
| **Distribuirana UI Proizvodnja** | Moč UI skoncentrirana v rokah redkih | Javno računsko omrežje, ekosistem odprtokodnih modelov, federirano UI sodelovanje |

## Načrt Mirnega Prehoda

### Faza 1: Ustavitev Krvavitve (2026–2028)
- Prepovedati popolnoma avtonomno smrtonosno orožje
- 50% "vojni davek" na presežne dobičke obrambnih izvajalcev
- Razkriti, kako deluje financiranje vojne
- Etični odbori za UI s klavzulo "ni za vojaško uporabo"

### Faza 2: Institucionalna Transformacija (2028–2035)
- Začetek veljavnosti Globalne Algoritmične Konvencije
- Vojaško-civilna industrijska konverzija (vodenje raket → precizna medicina, droni → pomoč ob nesrečah)
- 50% vojaških proračunov preusmerjenih v ekološko obnovo, UBI, razvoj Globalnega Juga
- Javni sklad za energijo, hrano, vodo in računalništvo

### Faza 3: Miroljubni Režim (2035–2050)
- Globalni vojaški proračuni zmanjšani na manj kot 1% BDP
- Avtonomno orožje globalno uničeno, trajna prepoved
- Pomanjkanje odpravljeno z UI-optimiziranim upravljanjem virov
- Vojna postane tako zastarela kot fevdalno dvobojevanje

## Poziv k Ukrepanju

**Vsem delavcem:** Organizirajte se — ustanovite ali se pridružite algoritmičnim sindikatom / digitalnim zadrugam. Zahtevajte transparentnost algoritmov. Prakticirajte alternative (uporabljajte odprtokodno UI, podpirajte zadružne platforme).

**Oblikovalcem politik:** Dajte prednost zakonodaji — podatki kot javni vir, obvezna ocena vpliva UI, minimalni standardi transparentnosti algoritmov. Investirajte v javno računsko infrastrukturo.

**Tehnologom:** Sistemi, ki jih gradite, bodo vplivali na milijarde. Ali vaš model osvobaja ljudi ali jih nadomešča? Ali vaš algoritem opolnomoča delavce ali iz njih črpa? **Tehnološke izbire nikoli niso nevtralne.**

**Državljanom sveta:** Razkrijte umetno pomanjkanje. Gradite nadnacionalno solidarnost. Sovražnik ni drug narod — to je sistem, ki profitira iz delitve in vojne.

Na ta prvomajski dan znova potrjujemo:

- Naj bodo podatki skupno bogastvo vsega človeštva, ne monopol redkih
- Naj prosti čas postane pravo merilo bogastva

**Prihodnost se ne napoveduje — ustvarja se.**
